

Необходимо разработать плагин, который будет отображать по дням информацию о количестве продаж по группам продуктов в виде столбчатой диаграммы Google Charts 

Таблицы с данными могут быть расположены как в локальной, так и удаленной базе данных.

Диаграмма должна отображаться в админ-панели WordPress в отдельном меню. Также плагин должен предоставлять shortcode для интеграции полученной диаграммы на фронт сайта.



К задаче прилагается sql-дамп, который содержит следующие таблицы с данными:


 1.Таблица orders - заказы, где:

o_creationdate		дата продажи
o_productid		id продукта



 2.Таблица products - список продуктов, где:

p_id			id продукта
p_productgroupid	id группы продуктов



 3.Таблица productgroups - группы продуктов, где:

pg_id			id группы продуктов
pg_supergroupid		id родительской группы
pg_nameshort		наименование группы
